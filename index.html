ipad-lightning-talks by maebert

A simple countdown app that will display the current speaker, time left, and the next speaker for a series of lightning talks.

Authors

Manuel Ebert (manuel.ebert@upf.edu)

Contact

Manuel Ebert (Maebert@UOS.de)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/maebert/ipad-lightning-talks

get the source code on GitHub : maebert/ipad-lightning-talks
